Spotify Wrapped Application

Spotify API, Node.JS, Python

This is a collaborative project I started with a group of my friends. Amongst them there are not only computer science students, but also aerospace and finance students from schools around the country. The functionality of this program is to interact with Spotify’s API to pull song and user listening data to show the user their top artists and songs. Using JavaScript and Python we have been able to pull data from Spotify and sift through it to find the useful information. And while it is still in progress, we’re a iming to complete it by the end of this year.
